### Properties of Boolean Operators

- All Boolean operators are commutative.  
- Most Boolean operators are associative, except NAND and NOR.

### XOR and XNOR (3 variables)

**Main identity:**
A ⊕ B ⊕ C = A ⊙ B ⊙ C

(where ⊙ = XNOR)

---

### Key relation

A ⊙ B = ¬(A ⊕ B)

---

### Correct proof

Start with:
(A ⊙ B) ⊙ C

= ¬( (A ⊙ B) ⊕ C )
= ¬( ¬(A ⊕ B) ⊕ C )

Now use the identity:
¬X ⊕ Y = ¬(X ⊕ Y)

So:
= ¬( ¬((A ⊕ B) ⊕ C) )
= (A ⊕ B) ⊕ C

= A ⊕ B ⊕ C

---

### Final result

A ⊕ B ⊕ C = A ⊙ B ⊙ C

✔ Proven

---

### General rule (parity insight)

* XOR = 1 when the number of 1s is **odd**
* XNOR = 1 when the number of 1s is **even**

So for n variables:

* If n is **odd**:
  A₁ ⊕ A₂ ⊕ ... ⊕ Aₙ = A₁ ⊙ A₂ ⊙ ... ⊙ Aₙ

* If n is **even**:
  A₁ ⊕ A₂ ⊕ ... ⊕ Aₙ = ¬(A₁ ⊙ A₂ ⊙ ... ⊙ Aₙ)

---

### Rule to remember

* Odd number of variables → XOR = chained XNOR
* Even number → XOR = NOT(chained XNOR)
