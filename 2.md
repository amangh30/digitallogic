## Truth Table for Two Variables (a, b)

We define:

* `f = a AND b`
* `~f = NOT f`
* `f_dual = a OR b` (dual of f)

| a | b | f (a AND b) | ~f (NOT f) | f_dual (a OR b) |
| - | - | ----------- | ---------- | --------------- |
| 0 | 0 | 0           | 1          | 0               |
| 0 | 1 | 0           | 1          | 1               |
| 1 | 0 | 0           | 1          | 1               |
| 1 | 1 | 1           | 0          | 1               |

### Notes

* `AND` is true only when both inputs are 1
* `NOT` flips the value
* The **dual** is obtained by replacing `AND` with `OR`

### Additional Note

- The number of Boolean functions of n variables for which ~f = f_dual is: 2^(2^n - 1)
